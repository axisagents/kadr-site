# KADR

> Every frame earns its place.

Marketing site for **Kadr** — a video editing studio for founders. Send raw footage (calls, podcasts, talking-head); get scroll-stopping short-form back in 48 hours.

## Stack
Single-file static site — `index.html` (vanilla HTML/CSS/JS, no build step). Fonts: Bricolage Grotesque + Geist Mono via Google Fonts.

## Develop
Open `index.html` in a browser, or serve locally:
```bash
npx serve .
```

## Deploy
Static deploy on Vercel — no build command, output dir is the repo root.
